# tujinwei.github.io

Personal blog and portfolio site for Jinwei Tu (David TU).

## Overview

This is a static site generated with [Hexo](https://hexo.io/) using the [Fluid](https://github.com/fluid-dev/hexo-theme-fluid) theme. It serves as a personal space for writing about software, technology, and other topics.

## Structure

```
├── 2021/          # Posts from 2021
├── 2022/          # Posts from 2022
├── 2024/          # Posts from 2024
├── about/         # About page
├── archives/      # Post archives
├── categories/    # Post categories
├── tags/          # Post tags
├── css/           # Stylesheets
├── js/            # JavaScript files
└── images/        # Image assets
```

## Deployment

The site is deployed via GitHub Pages and served at [tujinwei.github.io](https://tujinwei.github.io).
